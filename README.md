# Mergeta Menkir — Official Website

Traditional Ethiopian Healing Arts — Bilingual website (Amharic / English)

## 🌿 About
This is the official website for **Mergeta Menkir**, a traditional Ethiopian healing practice offering sacred arts and ancient knowledge services both within Ethiopia and internationally.

## 📁 Files
- `index.html` — Main website (single file, no dependencies except Google Fonts)

## 🚀 Deploying with GitHub Pages

1. Push this repo to GitHub (private or public)
2. Go to **Settings → Pages**
3. Under **Source**, select `main` branch → `/ (root)`
4. Click **Save**
5. Your site will be live at: `https://<your-username>.github.io/<repo-name>/`
